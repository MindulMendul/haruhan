# Data Structure Question Map

## 한 줄 정의

데이터를 어떤 형태로 저장하고 접근하느냐에 따라 시간·공간 복잡도가 달라지는 자료구조 질문 지도.

## 카드 요약

- **선형 구조**: 배열, 연결리스트, 스택, 큐.
- **우선순위 처리**: 우선순위 큐, 힙.
- **탐색/매핑**: 해시테이블, 그래프.
- **면접 핵심**: 삽입·삭제·탐색 비용과 실제 사용 사례를 같이 설명하기.

## 핵심 질문

- 자료구조
- 배열
- 연결리스트
- 스택
- 큐
- 우선순위 큐
- 힙
- 해시테이블
- 그래프

## 면접 포인트

1. **배열 vs 연결리스트**: 배열은 인덱스 접근이 빠르고, 연결리스트는 중간 삽입/삭제 구조가 다르다.
2. **스택/큐는 사용 상황이 중요하다**: DFS, BFS, 실행 취소, 작업 대기열 등 예시로 답한다.
3. **해시테이블은 평균과 최악이 다르다**: 충돌 처리와 해시 함수 품질을 같이 설명한다.

## 질문 예시

- “배열과 연결리스트의 차이는 무엇인가요?”
- “스택과 큐는 어디에 쓰이나요?”
- “해시 충돌은 어떻게 처리하나요?”
